# 🍴 Restaurant Food Ordering Website

## Overview

This is a simple Restaurant Food Ordering Website created using HTML, CSS, and JavaScript. It allows customers to select food items, enter their details, and calculate the total bill automatically.

## Features

* Restaurant header
* Food menu with prices
* Checkbox to select menu items
* Quantity input for each item
* Customer details form
* Automatic subtotal and grand total calculation
* Order summary display
* Customer details validation
* Thank You message after placing the order
* Basic responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Select food items, enter customer details, and click **Place Order** to view the bill.
